generate_column_file()

Generate the file for columns to be predicted (e.g., predict_phenotypes). This function takes into account the :min_genes property; does not request prediction of phenotypes which have fewer than min_genes genes in them.

[Source]

     # File app/models/john_predictor.rb, line 102
102:   def generate_column_file
103:     Dir.chdir(self.root) do
104:       if self.min_genes.nil? || self.min_genes == 0
105:         # Easy way -- just cut the cell file.
106:         `cut -f 2 #{self.predict_matrix.cell_file_path} |sort|uniq > #{self.column_filename}`
107:       else
108:         f = File.new(self.column_filename, "w")
109:         nrows_by_col = self.predict_matrix.number_of_rows_by_column
110:         nrows_by_col.each_pair do |col,nrows|
111:           f.puts(col) unless nrows < self.min_genes
112:         end
113:         f.close
114:       end
115:     end
116: 
117:     self.column_filename
118:   end
